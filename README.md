# 2. Project Info
<hr>

- 관광지 매칭 플랫폼 '어디까지 가봤니?' 서비스의 API 서버입니다.
- K-해커톤 최종 결선 프로젝트입니다.
- 서버 개발은 1인으로 진행 되었습니다.<br>
- K-해커톤 진행 과정 및 API 서버 제작 과정은 다음 <a href="https://blog.naver.com/sosow0212/222822588457">블로그</a>를 통해 순차적으로 확인하실 수 있습니다. (링크 누르시면, 같은 카테고리 내에 있습니다.)

<hr>

'어디까지 가봤니?' 오버투어리즘 현상 해결 및 여행의 다양화를 위해 고안된 프로젝트입니다.

지역주민은 관광객에게 본인 지역 가이딩을 제공할 수 있고, 또한 자신의 지역을 플랫폼에 홍보할 수 있습니다.

본 서비스를 통해 여행지 집중도가 분산되어, 일반 관광객들은 여러 지역을 알고 여행 갈 수 있습니다.
